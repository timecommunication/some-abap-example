### CDS Table Function

#### ZMAKT_CDS

一个简单的 CDS Table Function 定义，使用 ZCL_MATNR_FUNC_CDS 这个类来实现查询数据 。



#### ZPARAM_TFUNC

一个带参数的 Table Function , 使用 ZCL_TFUNC_AMDP_CLS 这个类来实现查询数据， 其中包含了 database function 的实现方式 和 database procedure 的实现方式 ，两者实现不一样 。



#### ZCDS_USE_TFUNC

基于 ZPARAM_TFUNC 这个 Table Function 定义的 CDS 视图
